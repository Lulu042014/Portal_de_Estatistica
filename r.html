Introdução a R

Introdução ao R

O que é o R?

R é uma linguagem de programação voltada para análise estatística e visualização de dados. É gratuita, de código aberto e amplamente utilizada em ciência de dados, estatística, bioinformática e pesquisa acadêmica.

Instalação e Ambiente de Trabalho

Instalar

  • R: https://cran.r-project.org
  • RStudio (IDE recomendada): https://posit.co/download/rstudio-desktop/

Principais arquivos

  • .R — script de código puro
  • .Rmd — R Markdown (mistura código + texto)
  • .qmd — Quarto (nova forma de relatórios interativos)

# Para os gráficos abaixo, simulamos um pequeno conjunto de dados de exemplo
set.seed(42)
dados <- data.frame(
  nome = paste0("Pessoa_", 1:20),
  sexo = sample(c("F", "M"), 20, replace = TRUE),
  idade = round(rnorm(20, mean = 28, sd = 6)),
  salario = round(rnorm(20, mean = 3500, sd = 800))
)
head(dados)
      nome sexo idade salario
1 Pessoa_1    F    36    3864
2 Pessoa_2    F    42    4064
3 Pessoa_3    F    20    4328
4 Pessoa_4    F    26    3013
5 Pessoa_5    M    27    3904
6 Pessoa_6    M    32    2126
# GRÁFICOS BÁSICOS
hist(dados$idade, main = "Histograma de Idade", xlab = "Idade", col = "skyblue")
boxplot(dados$idade ~ dados$sexo, main = "Idade por Sexo", xlab = "Sexo", ylab = "Idade")
plot(dados$idade, dados$salario, main = "Idade x Salário", xlab = "Idade", ylab = "Salário (R$)", pch = 19, col = "#2c3e50")

Interpretação

O histograma mostra a forma da distribuição de idade da amostra simulada, o boxplot compara essa distribuição entre os dois grupos de sexo, e o gráfico de dispersão explora se existe alguma relação visual entre idade e salário nos dados — nesse caso simulado, não esperamos padrão nenhum, já que os valores foram gerados de forma independente.

Controle de Fluxo

# Condicional
valores <- c(3, 12, 8)

if (any(valores > 10)) {
  print("Pelo menos um valor é maior que 10")
}
[1] "Pelo menos um valor é maior que 10"
if (all(valores > 10)) {
  print("Todos os valores são maiores que 10")
}

# Laço for
for (i in 1:5) {
  print(i^2)
}

[1] 4
