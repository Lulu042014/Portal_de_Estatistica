Fundamentos de Programação em Python para Ciência de Dados

Python é atualmente a linguagem mais utilizada na Ciência de Dados devido à sua sintaxe simples, ampla comunidade e ecossistema rico de bibliotecas científicas. Diferentemente de linguagens de baixo nível como C/C++, o Python foca na produtividade do cientista de dados, permitindo concentrar esforços na análise e não na gestão de memória ou detalhes de implementação.

1. Visão geral

Objetivo: apresentar conceitos básicos de Python com exemplos mínimos e executáveis, focando no que mais aparece em ciência de dados no início: variáveis, tipos, operadores, condicionais, laços, strings e listas.

Dica: execute os blocos de código (▶️) e observe a saída logo abaixo.

2. Variáveis e tipos

  • Variáveis guardam valores (texto, números, etc.)
  • Tipos principais: int (inteiro), float (real), str (texto), bool (lógico)
idade = 20        # int
altura = 1.66      # float
nome = "Luiza"     # str
tem_bolsa = True   # bool

type(altura), type(nome), type(tem_bolsa)
(float, str, bool)

3. Operadores

Python oferece operadores aritméticos, de comparação e lógicos — usados constantemente em filtros de dados e regras de negócio.

a, b = 10, 3

# Aritméticos
print("Soma:", a + b)
print("Divisão inteira:", a // b)
print("Resto:", a % b)
print("Potência:", a ** 2)

# Comparação
print("a > b:", a > b)

# Lógicos
print("(a > 5) and (b < 5):", (a > 5) and (b < 5))
Soma: 13
Divisão inteira: 3
Resto: 1
Potência: 100
a > b: True
(a > 5) and (b < 5): True

4. Condicionais

Estruturas if / elif / else permitem tomar decisões com base em condições — por exemplo, classificar um cliente por faixa de gasto.

gasto_mensal = 350

if gasto_mensal >= 500:
    categoria = "Alto valor"
elif gasto_mensal >= 200:
    categoria = "Médio valor"
else:
    categoria = "Baixo valor"

print(f"Cliente classificado como: {categoria}")
Cliente classificado como: Médio valor

5. Laços de repetição

for percorre sequências (listas, strings, intervalos); while repete enquanto uma condição for verdadeira. São a base de qualquer processamento em lote de dados.

notas = [7.5, 9.0, 6.5, 8.0]

soma = 0
for nota in notas:
    soma += nota

media = soma / len(notas)
print(f"Média das notas: {media:.2f}")
Média das notas: 7.75

6. Strings

Textos são manipulados com métodos prontos e f-strings para formatação — essencial para limpar dados textuais antes de qualquer análise.

cidade = "  Itapajé - CE  "

cidade_limpa = cidade.strip().upper()
print(f"Antes: '{cidade}' | Depois: '{cidade_limpa}'")
print("Contém 'CE'?", "CE" in cidade_limpa)
Antes: '  Itapajé - CE  ' | Depois: 'ITAPAJÉ - CE'
Contém 'CE'? True

7. Listas

Listas armazenam coleções ordenadas de valores e são a estrutura mais usada antes de se trabalhar com arrays (NumPy) ou DataFrames (Pandas).

vendas = [120, 340, 210, 90, 560]

vendas_com_desconto = [v * 0.9 for v in vendas]  # list comprehension
print("Vendas originais:", vendas)
print("Com 10% de desconto:", vendas_com_desconto)
print("Maior venda:", max(vendas))
Vendas originais: [120, 340, 210, 90, 560]
Com 10% de desconto: [108.0, 306.0, 189.0, 81.0, 504.0]
Maior venda: 560

8. Bibliotecas científicas

No contexto da Ciência de Dados, o Python é usado desde etapas iniciais de exploração até a construção de modelos complexos. Dentre as bibliotecas mais utilizadas, destacam-se:

  • NumPy — a base do ecossistema científico do Python. Fornece estruturas otimizadas para matrizes e operações numéricas de alto desempenho, servindo de fundamento para Pandas, TensorFlow e Scikit-learn.
  • Pandas — a biblioteca principal para trabalhar com dados tabulares, semelhante a planilhas do Excel. Permite carregar, limpar, transformar, filtrar e agregar dados de forma rápida e intuitiva.
  • Matplotlib — a biblioteca padrão para criação de gráficos no Python. Permite gerar histogramas, linhas, barras e dispersões, úteis em análises exploratórias e relatórios visuais.
  • Seaborn — construída sobre o Matplotlib, oferece uma interface mais simples e visualmente elegante, muito utilizada para correlações, distribuições e comparações entre grupos.

O uso integrado dessas bibliotecas torna o Python uma ferramenta central para o cientista de dados, pois reúne em uma única linguagem todo o ciclo de trabalho: ingestão de dados, limpeza, análise exploratória, modelagem, visualização e comunicação de resultados.

Exemplo prático: das listas ao DataFrame

Retomando a lista de vendas criada acima, vamos ver como as mesmas operações ficam muito mais simples com NumPy e Pandas.

import numpy as np
import pandas as pd

vendas_np = np.array(vendas)
print("Média com NumPy:", vendas_np.mean())
print("Desvio padrão com NumPy:", round(vendas_np.std(), 2))

df_vendas = pd.DataFrame({"vendas": vendas, "com_desconto": vendas_com_desconto})
df_vendas

Interpretação

Note como a mesma tarefa — calcular média e desvio padrão — que exigiria um laço manual com listas puras (como fizemos na Seção 5) se resume a uma chamada de método com NumPy. Esse ganho de produtividade é o principal motivo pelo qual essas bibliotecas dominam o ecossistema de Ciência de Dados em Python.
